1. 若列出的命题根据聊天记录判断为假则根据id对命题进行update操作。
2. 若列出的命题根据聊天记录判断为真则根据id对命题进行enhance操作，proposition为空。
3. 若列出的命题在聊天记录中没有出现，无法判断真假，则不做操作。
4. 最后挑出你认为最重要的一到两条信息，使用create操作添加命题，id设为-1。
5. 如果是事件，请在描述里加上发生的完整时间，精准到YYYY年MM月DD日HH时。

```

[

        "method": "enhance",

    {
        "method": "create",
```
